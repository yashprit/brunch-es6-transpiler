> brunch plugin for es6 transpiler


## Install

```sh
$ npm install --save brunch-es6-transpiler
```


## Usage

```js
var brunch-es6-transpiler = require('brunch-es6-transpiler');

brunch-es6-transpiler('Rainbow');
```
